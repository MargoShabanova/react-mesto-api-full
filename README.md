# react-mesto-api-full
Репозиторий для приложения проекта `Mesto`, включающий фронтенд и бэкенд части приложения со следующими возможностями: авторизации и регистрации пользователей, создание и удаление карточек с фотографиями, редактирование информации о пользователе, просмотр фотографии из карточки в увеличенном формате.

Бэкенд расположен в директории `backend/`, а фронтенд - в `frontend/`.
